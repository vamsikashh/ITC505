Mad Lib Form






Project Addendum

Overview: In this lab, I implemented a Mad Lib web form using ExpressJS. The form captures user inputs and generates a fun story based on those inputs. The project consists of an HTML form for input collection, a server-side script to handle form submissions, and a CSS file for styling the form.

Steps Followed:

  1. Setup and Installation: Installed Node.js and npm using Homebrew on macOS. Initialized a new Node.js project using npm init -y. Installed required npm packages: express, morgan, and forever.
  2. Project Structure: Created the project directory and set up the following structure: 
    ├── public/
│   ├── index.html
│   ├── style.css
├── server.js
├── package.json
└── package-lock.json
  3. Server Configuration: Created server.js with the following functionalities:
    • Configured ExpressJS to serve static files from the public directory.
    • Implemented a POST route to handle form submissions and generate the Mad Lib story.
  4. HTML Form: Designed an HTML form (index.html) with five input fields (noun, verb, adjective, adverb, and plural noun) and a submit button. Added a footer to display the last modified date using JavaScript.
  5. CSS Styling: Created style.css to style the HTML form, making it visually appealing and user-friendly.
  6. Form Handling and Feedback: Implemented form handling in server.js to generate and display the Mad Lib story based on user inputs.
